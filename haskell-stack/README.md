# $project_name by $author

Haskell application with main executable only built using Stack

## Clone repository

```
git clone $public_git_repo:$git_user_name/$project_name.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build
```

## Run application

```
stack exec $project_name-app
```

## Run tests

```
stack test
```
